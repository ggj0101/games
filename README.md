# GAMES

Vue 3 + TypeScript + Vite 的小遊戲合集專案。

## AI / Spec-driven workflow

本 repo 已啟用 **OpenSpec**（spec-driven development）並支援：
- Codex（`.codex/`）
- Gemini CLI（`.gemini/`）
- GitHub Copilot（`.github/`）

工作方式：
- 變更流程（OpenSpec）：看 `ai/WORKFLOW.md`
- Vue/TS 實作規範（skills 風格）：看 `ai/VUE_SKILLS.md`
- 專案工程共識（cleanup/dynamic import 等）：看 `ai/PROJECT_CONVENTIONS.md`

產品/遊戲規格庫仍維持在：`spec/`
